# Typing Speed Test

A browser-based typing speed test with real-time WPM, accuracy tracking, and instant visual feedback.

## Features

- **Live WPM & accuracy** — calculated dynamically as you type
- **Color-coded feedback** — green for correct, red for incorrect characters
- **Randomized quotes** — fresh content for each session
- **Instant reset** — one-click new test
- **Responsive** — works on all screen sizes

## Tech Stack

React 18, TypeScript, Vite 5, Tailwind CSS 3, Lucide Icons, Docker

## Quick Start

```bash
npm install
npm run dev

# Docker
docker compose up -d --build
```

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run lint` — lint
- `npm run typecheck` — TypeScript check
